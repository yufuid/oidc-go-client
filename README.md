# oidc-go-client
An OpenID Connect reference implementation in Golang

## Getting started
First clone the repository:

    git clone https://github.com/yufuid/oidc-go-client.git
    cd oidc-go-client
 
To start client run:

     go run cmd/main.go server --config config/config.yml
